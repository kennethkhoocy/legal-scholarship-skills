\begin{abstract}
This fixture tests custom enumeration labels, nested alphabetic labels, and a
resumed enumeration sequence using the \texttt{enumitem} package.
\end{abstract}

\section{Protocol Stages}
\label{sec:protocol}

The review protocol is expressed as a numbered sequence with roman numerals.
The first block records screening tasks:
\begin{enumerate}[label=(\roman*),series=review,leftmargin=*]
\item verify that the intake form contains a site identifier;
\item classify the record into the initial eligibility stratum;
\item record the reviewer initials in the audit log.
\end{enumerate}

The third item includes a nested list with alphabetic labels:
\begin{enumerate}[label=\alph*),leftmargin=2em]
\item initials must match the reviewer roster;
\item duplicate initials are resolved by the supervisor code.
\end{enumerate}

\section{Resumed Sequence}
\label{sec:resumed}

After the nested notes, the main review sequence resumes from the previous roman
number:
\begin{enumerate}[resume=review,label=(\roman*),leftmargin=*]
\item reconcile flagged records against the supervisor notes;
\item mark the final analytic status before outcome data are inspected.
\end{enumerate}
